# 🌍 Ecosphere

> A free, AI-powered environmental data analytics platform

![Python](https://img.shields.io/badge/Python-3.x-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![SDG13](https://img.shields.io/badge/SDG-13%20Climate%20Action-orange)

## 🚀 Features
- 🌬️ Real-time global air quality monitoring
- 🗑️ AI waste classification from photos
- 🌱 Personal carbon footprint calculator
- 📊 ML-powered AQI predictions

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | Python, Flask |
| Frontend | React |
| Database | SQLite |
| AI/ML | Scikit-learn, TensorFlow |
| Data | OpenAQ API |

## 📦 Installation
```bash
pip install -r requirements.txt
python fetch_aqi.py
python ml_model.py
```
